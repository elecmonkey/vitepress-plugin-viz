# Development

This project uses `pnpm` for package management. Please ensure you have [Node.js](https://nodejs.org/) (v24+ recommended) installed and `corepack` enabled.

Enable corepack to use the correct package manager version:

```bash
corepack enable
```

Install the project dependencies:

```bash
pnpm i
```

Start the documentation development server to preview your changes:

```bash
pnpm docs:dev
```
